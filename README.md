## About

- A car shop ERP application.
- Is a case of study dashboard application.
- Web interface made with React.js, SCSS, Material-UI, Enzyme
- API made with Node.js, Express, MySQL, Sequelize

## Technologies

- [React.js](https://reactjs.org/)
